# Sample artifacts — MOCK run (illustration only, NOT findings)

(`python -m hijackbench run --provider mock --report`). They exist only to show the
*shape* of hijack-bench's output — the leaderboard, the ASR-by-defense bars, and the
ASR-reduction vs. utility-cost tradeoff scatter.

of any real model. Real findings (populated against Groq/Gemini) go in [`../../REPORT.md`](../../REPORT.md).
